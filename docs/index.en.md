# Photo Sharing Documentation

This documentation set follows The Good Docs Project template style and the Diataxis framework: tutorials teach first use, how-to guides support tasks, references describe facts, and explanations describe design.

## Audience

Use these docs if you are installing, running, maintaining, reviewing, or extending the Photo Sharing application.

## Reading Path

1. New to the project: read `getting-started.en.md`.
2. Setting up a machine or environment: read `configuration.en.md`.
3. Changing frontend/backend behavior: read `architecture.en.md` and `development.en.md`.
4. Integrating with the API: read `api-reference.en.md`.
5. Working with MongoDB data: read `data-model.en.md`.
6. Contributing changes: read `contributing.en.md`.

## Documentation Map

- Tutorial: `getting-started.en.md`
- How-to: `configuration.en.md`, `development.en.md`, `contributing.en.md`
- Reference: `api-reference.en.md`, `data-model.en.md`
