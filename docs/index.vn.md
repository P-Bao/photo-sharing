# Tài liệu Photo Sharing

Bộ tài liệu này dùng phong cách template của The Good Docs Project và cấu trúc Diataxis: tutorial giúp người mới chạy được dự án, how-to hỗ trợ tác vụ, reference mô tả dữ kiện kỹ thuật, explanation giải thích thiết kế.

## Đối tượng đọc

Dùng tài liệu này khi bạn cần cài đặt, chạy, bảo trì, review hoặc mở rộng ứng dụng Photo Sharing.

## Lộ trình đọc

1. Mới bắt đầu với dự án: đọc `getting-started.vn.md`.
2. Cấu hình máy hoặc môi trường: đọc `configuration.vn.md`.
3. Thay đổi frontend/backend: đọc `architecture.vn.md` và `development.vn.md`.
4. Tích hợp với API: đọc `api-reference.vn.md`.
5. Làm việc với dữ liệu MongoDB: đọc `data-model.vn.md`.
6. Đóng góp thay đổi: đọc `contributing.vn.md`.

## Bản đồ tài liệu

- Tutorial: `getting-started.vn.md`
- How-to: `configuration.vn.md`, `development.vn.md`, `contributing.vn.md`
- Reference: `api-reference.vn.md`, `data-model.vn.md`
- Explanation: `architecture.vn.md`
