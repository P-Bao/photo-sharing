# Frontend Quick Reference

The frontend is a React application for browsing users, viewing photos, posting comments, registering, logging in, and uploading photos.

## Run

```powershell
npm install
npm start
```

The app runs at `http://localhost:3000`.

API calls are made through `src/lib/fetchModelData.js`. The current code uses a hard-coded backend URL.
