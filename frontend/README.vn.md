# Frontend Quick Reference

Frontend là ứng dụng React để xem users, xem ảnh, bình luận, đăng ký, đăng nhập và upload ảnh.

## Chạy

```powershell
npm install
npm start
```

Ứng dụng chạy tại `http://localhost:3000`.

Các lời gọi API đi qua `src/lib/fetchModelData.js`. Code hiện tại đang dùng backend URL được hard-code.
